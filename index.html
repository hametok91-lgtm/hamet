Publicar un sitio web en GitHub es un proceso sencillo que se hace usando el servicio llamado GitHub y su herramienta GitHub Pages. Primero debes crear una cuenta y luego un repositorio nuevo, que es como una carpeta en línea donde subirás los archivos de tu sitio (HTML, CSS, JavaScript, imágenes, etc.). Después, subes todos los archivos de tu página al repositorio, ya sea arrastrándolos desde tu computadora o usando Git si sabes manejarlo. Es importante que el archivo principal se llame index.html, porque ese será el que se mostrará primero. Cuando ya estén los archivos en el repositorio, entras a la configuración (Settings), buscas la sección “Pages” y activas GitHub Pages seleccionando la rama principal (normalmente “main”). Guardas los cambios y, en pocos minutos, GitHub te dará un enlace público donde tu sitio web estará disponible para cualquier persona. Así, sin necesidad de pagar hosting, puedes tener tu página publicada en internet de forma gratuita y fácil.
